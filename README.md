# Dotfiles

Macbook のセットアップ

## セットアップ手順

### 1. システム環境設定を変更する

左上の ` > システム環境設定...` から以下の設定を行う。

#### 一般

| 項目         | 値         |
|------------|-----------|
| 外観モード      | ダーク       |
| スクロールバーの表示 | スクロール時に表示 |

#### デスクトップとスクリーンセーバ

| 項目                    | 値        |
|-----------------------|----------|
| スクリーンセーバ              | Monterey |
| スクリーンセーバ > 開始までの時間    | 5分       |
| スクリーンセーバ > 時計と一緒に表示   | オン       |
| スクリーンセーバ > ホットコーナー... | 全てなし     |

#### Dock とメニューバー

| 項目                                       | 値         |
|------------------------------------------|-----------|
| Dock とメニューバー > ウインドウをしまうときのエフェクト         | スケールエフェクト |
| Dock とメニューバー > ウインドウタイトルバーのダブルクリックで      | オフ        |
| Dock とメニューバー > Dock を自動的に表示/非表示          | オン        |
| Dock とメニューバー > 最近使ったアプリケーションを Dock に表示   | オフ        |
| Dock とメニューバー > デスクトップにメニューバーを自動的に表示/非表示  | オン        |
| Dock とメニューバー > フルスクリーンでメニューバーを自動的に表示/非表示 | オン        |
| Bluetooth > メニューバーに表示                    | オフ        |
| AirDrop > メニューバーに表示                      | オフ        |
| バッテリー > 割合（%）を表示                         | オン        |
| 時計 > 秒を表示                                | オン        |
| Spotlight > メニューバーに表示                    | オフ        |
| Siri > メニューバーに表示                         | オフ        |

#### キーボード

| 項目                          | 値    |
|-----------------------------|------|
| キーボード > キーのリピート             | 最も速い |
| キーボード > リピート入力認識までの時間       | 最も短い |
| ユーザ辞書 > 英字入力中にスペルを自動変換      | オフ   |
| ユーザ辞書 > 文頭を自動的に大文字にする       | オフ   |
| ユーザ辞書 > スペースバーを2回押してピリオドを入力 | オフ   |
| ユーザ辞書 > Touch Bar に入力候補を表示  | オフ   |
| ユーザ辞書 > スマート引用符とスマートダッシュを使用 | オフ   |

#### トラックパッド

| 項目                          | 値    |
|-----------------------------|------|
| ポイントとクリック > 調べる&データ検出       | オフ   |
| ポイントとクリック > タップでクリック        | オン   |
| ポイントとクリック > 軌跡の速さ           | 最も速い |
| スクロールとズーム > スマートズーム         | オフ   |
| その他のジェスチャ > ページ間をスワイプ       | オフ   |
| その他のジェスチャ > 通知センター          | オフ   |
| その他のジェスチャ > アプリケーション Expose | オフ   |
| その他のジェスチャ > デスクトップを表示       | オフ   |

### 2. Homebrew でセットアップする

まず、以下のコマンドで Homebrew をインストールする。

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

`.Brewfile` を用意し、必要なライブラリを一括でインストールする。

```bash
touch .Brewfile
```

### 3. ログインシェルを Bash に変更する

`/etc/shells` を以下のように書き換えて、Homebrew でインストールした Bash にパスを通す。

```bash
sudo nano /etc/shells 
```

```text
# List of acceptable shells for chpass(1).
# Ftpd will not allow users to connect who are not using
# one of these shells.

/bin/bash
/bin/csh
/bin/dash
/bin/ksh
/bin/sh
/bin/tcsh
/bin/zsh
/usr/local/bin/bash
```

その後に以下のコマンドでログインシェルを Bash に変更する。

```bash
chsh -s /usr/local/bin/bash
```
