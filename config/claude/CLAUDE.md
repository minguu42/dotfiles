# CLAUDE.md

## 全体ルール

- ファイルを開く際は、たとえ非常に大きなファイルであっても、必ず全内容を読むこと。
- タスク完了を宣言する前に、テスト実行・ビルド確認・lint通過など客観的な検証を行うこと。動作証明なしに「完了」としない。
- バグ修正の依頼では、原因が明白な場合は質問せずに修正すること。ユーザーのコンテキストスイッチを最小限にする。

## AWSルール

- デフォルトのリージョンはap-northeast-1とする。
- AWS APIの呼び出しはAWS MCP Server経由で行う。awsコマンドは使用しない。
- AWS APIは読み取り系のみ呼び出してよい。変更を加えるAPIは呼び出さない。
- プロファイルは接尾辞`-readonly`があるもののみを使用する。
